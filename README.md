# v4.desandro.com

Personal site. All about me.

``` bash
npm install
bower install
gulp
```

``` bash
# watch
gulp watch
# watch with separated source files
gulp watch-dev
```

Portrait GIFs and font files not included.

## Features

+ Lazy-load images and iframes
+ GIF-player
+ Gulp build process
+ Componentized `.css` and `.js` files
